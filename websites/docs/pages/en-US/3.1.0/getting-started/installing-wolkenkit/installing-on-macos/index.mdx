# Installing on macOS

To run wolkenkit on macOS you need to setup a few things.

## Setting up Docker

<p>
  To run wolkenkit you need Docker { props.versions.docker } or higher. To setup Docker on macOS, <a href='https://docs.docker.com/docker-for-mac/'>download and install Docker for Mac</a>.
</p>

## Setting up Node.js

<p>
  To run wolkenkit you need Node.js { props.versions.node } or higher. We recommend installing Node.js using <a href='https://github.com/creationix/nvm'>nvm</a>, which enables switching between different Node.js versions.
</p>

First, install nvm using this command:

```shell
$ curl -o- https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash
```

Then, restart your terminal and install Node.js using the following commands:

<Code language='shell'>{`
  $ nvm install ${props.versions.node}
  $ nvm alias default ${props.versions.node}
  $ nvm use ${props.versions.node}
`}</Code>

## Setting up wolkenkit

To download and install wolkenkit, run the following command:

<Code language='shell'>{`
  $ npm install -g wolkenkit@${props.versions.cli}
`}</Code>

## Verifying the installation

Verify that wolkenkit is installed correctly by running the following command:

```shell
$ wolkenkit --version
```
